Справку по использованию можно получить с помощью команды --help. Вкратце, есть два режима: прогрузить все страницы
достижимые из заданной и находящиеся не более чем на глубине --depth (по умолчанию 2). И второй режим получить -n страниц,
который достижимы из указанной (необязательно непосредственно).

Дополнительно устанавливать какие-либо модули не требуется, проект разрабатывался и тестировался на версии питона 3.6
